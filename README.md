# Dominos Pizza Sales Dashboard (Tableau)

## Overview
This project is a Tableau dashboard created to analyze Dominos Pizza sales data.
The dashboard shows overall sales performance, top pizzas, pizza categories,
ingredients usage, and order distribution by part of the day.
